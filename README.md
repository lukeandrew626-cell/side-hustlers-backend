# Side Hustlers

AI-powered app that helps gig drivers (DoorDash, Uber Eats, Instacart) instantly decide whether to take or skip an order based on payout, miles, and time.

## Features
- Screenshot-based order analysis
- Instant TAKE / SKIP / MAYBE recommendation
- Supports DoorDash, Uber Eats, Instacart
- Fast decision-making for real-time orders

## Tech Stack
- Frontend: React Native (Expo)
- Backend: Node.js
- AI: OpenAI
